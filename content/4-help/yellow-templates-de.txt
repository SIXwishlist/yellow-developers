Du kannst die Funktionalität deiner Webseite anpassen.

## Templates

Templates bestimmen die Funktionalität deiner Webseite. Das Standardtemplate wird in der Datei `system/config/config.ini` festgelegt. Eine anderes Template lässt sich in den [Einstellungen](markdown-cheat-sheet-de#einstellungen) ganz oben auf jeder Seite festlegen, zum Beispiel `Template: default`. 

Hier ist eine Beispiel-Templatedatei `system/themes/templates/default.html`:

``` php
<?php $yellow->snippet("header") ?>
<?php $yellow->snippet("content-sidebar") ?>
<?php $yellow->snippet("content-main") ?>
<?php $yellow->snippet("footer") ?>
```

Snippets sind die Bausteine deiner Webseite. Es sind kleine Stücke [PHP](http://www.w3schools.com/php/) um [HTML](http://www.w3schools.com/html/) zu erstellen. Du kannst alles in ein Template schreiben, aber es ist häufig praktischer ein Template in mehrere Snippets aufzuteilen und diese wiederzubenutzen.

Hier ist eine Beispiel-Snippetdatei `system/themes/snippets/content-main.php`:

``` php
<div class="content main">
<h1><?php echo $yellow->page->getHtml("titleContent") ?></h1>
<?php echo $yellow->page->getContent() ?>
</div>
```

## Template anpassen

Um ein anderes Template zu benutzen, legst du das `Template` ganz oben auf einer Seite fest. Du kannst auch ein Template mit dem Namen eines `content`-Verzeichnisses erstellen und es wird automatisch benutzt. Das ermöglicht es beliebige Layouts mit Yellow zu erzeugen.

Hier ist eine Beispiel-Templatedatei `system/themes/templates/home.html`:

``` php
<?php $yellow->snippet("header") ?>
<?php $yellow->snippet("content-sidebar") ?>
<?php $yellow->snippet("content-home") ?>
<?php $yellow->snippet("footer") ?>
```

Hier ist eine Beispiel-Snippetdatei `system/themes/snippets/content-home.php`:

``` php
<div class="content main">
<h1><?php echo $yellow->page->getHtml("titleContent") ?></h1>
<?php echo $yellow->page->getContent() ?>
<?php /* Add more PHP code here */ ?>
</div>
```

Hier ist die selbe Snippetdatei mit Blogeinträgen:

``` php
<div class="content main">
<h1><?php echo $yellow->page->getHtml("titleContent") ?></h1>
<?php echo $yellow->page->getContent() ?>
<?php $pages = $yellow->pages->index()->filter("template", "blog")->sort("published", false)->limit(3) ?>
<?php foreach($pages as $page): ?>
<h2><a href="<?php echo $page->getLocation(true) ?>"><?php echo $page->getHtml("title") ?></a></h1>
<p><?php echo $yellow->toolbox->createTextDescription($page->getContent(), 250) ?></p>
<?php endforeach ?>
<?php $yellow->page->setLastModified($pages->getModified()) ?>
<?php $yellow->page->setHeader("Cache-Control", "max-age=60") ?>
</div>
```

## Navigation anpassen

Um die Navigation anzupassen, öffne die Datei `system/config/config.ini` und ändere `Navigation`. Eine andere Navigation lässt sich in den [Einstellungen](markdown-cheat-sheet-de#einstellungen) ganz oben auf jeder Seite festlegen, zum Beispiel `Navigation: navigation`. 

Die folgenden Navigationen sind vorhanden:

`Navigation: navigation` zeigt Hauptseiten  
`Navigation: navigation-sidebar` zeigt Unterseiten  
`Navigation: navigation-tree` zeigt Dropdown-Menü   
`Navigation: navigation-search` zeigt [Suchzeile](https://github.com/datenstrom/yellow-plugins/tree/master/search)  
`Navigation: navigation-multilanguage` zeigt [Sprachauswahl](language-configuration-de#mehrsprachen-modus)   

Hier ist eine Beispiel-Snippetdatei `system/themes/snippets/navigation.php`:

``` php
<?php $pages = $yellow->pages->top() ?>
<?php $yellow->page->setLastModified($pages->getModified()) ?>
<div class="navigation">
<ul>
<?php foreach($pages as $page): ?>
<li><a<?php echo $page->isActive() ? " class=\"active\"" : "" ?> href="<?php echo $page->getLocation(true) ?>"><?php echo $page->getHtml("titleNavigation") ?></a></li>
<?php endforeach ?>
</ul>
</div>
<div class="navigation-banner"></div>
```

Hier ist die selbe Snippetdatei mit [Icons für soziale Medien](https://github.com/datenstrom/yellow-plugins/tree/master/fontawesome):

``` php
<?php $pages = $yellow->pages->top() ?>
<?php $yellow->page->setLastModified($pages->getModified()) ?>
<div class="navigation">
<ul>
<?php foreach($pages as $page): ?>
<li><a<?php echo $page->isActive() ? " class=\"active\"" : "" ?> href="<?php echo $page->getLocation(true) ?>"><?php echo $page->getHtml("titleNavigation") ?></a></li>
<?php endforeach ?>
<li><a href="https://twitter.com/datenstromse"><i class="fa fa-twitter"></i></a></li>
<li><a href="https://github.com/datenstrom"><i class="fa fa-github"></i></a></li>
<li><a href="https://instagram.com/datenstromse"><i class="fa fa-instagram"></i></a></li>
</ul>
</div>
<div class="navigation-banner"></div>
```

## Fußzeile anpassen

Um die Fußzeile anzupassen, ändere das entsprechende Snippet. Die Fußzeile wird ganz unten auf allen Seiten angezeigt. Du kannst beispielsweise Informationen über deine Webseite und Links anzeigen.

Hier ist eine Beispiel-Snippetdatei `system/themes/snippets/footer.php`:

``` php
<div class="footer">
<div class="siteinfo">
<a href="<?php echo $yellow->page->base."/" ?>">&copy; 2016 <?php echo $yellow->page->getHtml("sitename") ?></a>.
<a href="<?php echo $yellow->page->get("pageEdit") ?>">Edit</a>.
<a href="<?php echo $yellow->text->get("yellowUrl") ?>">Made with Yellow</a>.
</div>
</div>
</div>
<?php echo $yellow->page->getExtra("footer") ?>
</body>
</html>
```

Hier ist die selbe Snippetdatei mit Links zu soziale Medien:

``` php
<div class="footer">
<div class="siteinfo">
<a href="https://twitter.com/datenstromse">Twitter</a>.
<a href="https://github.com/datenstrom">GitHub</a>.
<a href="https://instagram.com/datenstromse">Instagram</a>.
<a href="<?php echo $yellow->page->get("pageEdit") ?>">Edit</a>.
<a href="<?php echo $yellow->text->get("yellowUrl") ?>">Made with Yellow</a>.
</div>
</div>
</div>
<?php echo $yellow->page->getExtra("footer") ?>
</body>
</html>
```

## Sidebar

Um eine Sidebar zu erzeugen, erstelle die Datei `sidebar.txt` in einem beliebigem `content`-Verzeichnis. Die Sidebar wird automatisch auf allen Seiten im gleichen Verzeichnis angezeigt. Eine andere Sidebar lässt sich in den [Einstellungen](markdown-cheat-sheet-de#einstellungen) ganz oben auf jeder Seite festlegen.

Hier ist eine Beispiel-Sidebar `content/1-home/sidebar.txt`:

```
---
Title: Sidebar
Status: hidden
---
Links

* [Twitter](https://twitter.com/datenstromse)
* [GitHub](https://github.com/datenstrom)
* [Instagram](https://instagram.com/datenstromse)
```
