Du kannst den Quellcode deiner Webseite anpassen. [Demo anschauen](/tests/custom-template).

## Templates

Templates erzeugen den Quellcode deiner Webseite. Das Standardtemplate wird in den [Systemeinstellungen](adjusting-system-de#systemeinstellungen) festgelegt. Eine anderes Template lässt sich in den [Einstellungen](markdown-cheat-sheet-de#einstellungen) ganz oben auf jeder Seite festlegen, zum Beispiel `Template: default`.

Hier ist eine Beispiel-Templatedatei `system/themes/templates/default.html`:

``` php
<?php $yellow->snippet("header") ?>
<?php $yellow->snippet("content-default") ?>
<?php $yellow->snippet("footer") ?>
```

Snippets sind die Bausteine deiner Webseite. Es sind kleine Stücke [HTML](http://www.w3schools.com/html/) und [PHP](http://www.w3schools.com/php/). Du kannst alles in ein Template schreiben, aber es ist häufig praktischer ein Template in mehrere Snippets aufzuteilen und wiederzubenutzen. Mache was am Besten für dich funktioniert.

## Template anpassen

Um den Quellcode deiner Webseite anzupassen, ändere das entsprechende Template. Das Template legt fest welche Snippets benutzt werden. Du kannst entweder die vorhandenen Snippets ändern oder ein neues Template anlegen, zum Beispiel `Template: custom`.

Hier ist eine Beispiel-Templatedatei `system/themes/templates/custom.html`:

``` php
<?php $yellow->snippet("header") ?>
<?php $yellow->snippet("content-custom") ?>
<?php $yellow->snippet("footer") ?>
```

Hier ist eine Beispiel-Snippetdatei `system/themes/snippets/content-custom.php`:

``` php
<div class="content">
<?php $yellow->snippet("sidebar") ?>
<div class="main">
<h1><?php echo $yellow->page->getHtml("titleContent") ?></h1>
<?php echo $yellow->page->getContent() ?>
<!-- Add more HTML and PHP code here -->
</div>
</div>
```

## Navigation anpassen

Um die Navigation anzupassen, ändere das entsprechende Snippet. Die Standardnavigation wird in den [Systemeinstellungen](adjusting-system-de#systemeinstellungen) festgelegt. Eine andere Navigation lässt sich in den [Einstellungen](markdown-cheat-sheet-de#einstellungen) ganz oben auf jeder Seite festlegen, zum Beispiel `Navigation: navigation`. 

[image screenshot-template1.png Screenshot]

Die folgenden Navigationen sind vorhanden:

`Navigation: navigation` zeigt Hauptseiten  
`Navigation: navigation-sidebar` zeigt Unterseiten  
`Navigation: navigation-tree` zeigt alle Seiten  
`Navigation: navigation-search` zeigt [Suchzeile](https://github.com/datenstrom/yellow-plugins/tree/master/search)  
`Navigation: navigation-multilanguage` zeigt [Sprachauswahl](language-configuration-de#mehrsprachen-modus)   

Hier ist eine Beispiel-Snippetdatei `system/themes/snippets/navigation.php`:

``` php
<?php $pages = $yellow->pages->top() ?>
<?php $yellow->page->setLastModified($pages->getModified()) ?>
<div class="navigation">
<ul>
<?php foreach($pages as $page): ?>
<li><a<?php echo $page->isActive() ? " class=\"active\"" : "" ?> href="<?php echo $page->getLocation(true) ?>"><?php echo $page->getHtml("titleNavigation") ?></a></li>
<?php endforeach ?>
</ul>
</div>
<div class="navigation-banner"></div>
```

Hier ist die selbe Snippetdatei mit [Icons für soziale Medien](https://github.com/datenstrom/yellow-plugins/tree/master/fontawesome):

``` php
<?php $pages = $yellow->pages->top() ?>
<?php $yellow->page->setLastModified($pages->getModified()) ?>
<div class="navigation">
<ul>
<?php foreach($pages as $page): ?>
<li><a<?php echo $page->isActive() ? " class=\"active\"" : "" ?> href="<?php echo $page->getLocation(true) ?>"><?php echo $page->getHtml("titleNavigation") ?></a></li>
<?php endforeach ?>
<li><a href="https://twitter.com/datenstromse"><i class="fa fa-twitter fa-lg"></i></a></li>
<li><a href="https://github.com/datenstrom"><i class="fa fa-github fa-lg"></i></a></li>
<li><a href="https://instagram.com/datenstromse"><i class="fa fa-instagram fa-lg"></i></a></li>
</ul>
</div>
<div class="navigation-banner"></div>
```

## Fußzeile anpassen

Um die Fußzeile anzupassen, ändere das entsprechende Snippet. Das Template legt fest welche Snippets benutzt werden, die Fußzeile befindet sich normalerweise ganz unten. Du kannst entweder das vorhandene Snippet ändern oder ein neues Snippet anlegen.

[image screenshot-template2.png Screenshot]

Hier ist eine Beispiel-Snippetdatei `system/themes/snippets/footer.php`:

``` php
<div class="footer">
<div class="siteinfo">
<a href="<?php echo $yellow->page->base."/" ?>">&copy; 2017 <?php echo $yellow->page->getHtml("sitename") ?></a>.
<a href="<?php echo $yellow->page->get("pageEdit") ?>">Edit</a>.
<a href="<?php echo $yellow->text->get("yellowUrl") ?>">Made with Yellow</a>.
</div>
</div>
</div>
<?php echo $yellow->page->getExtra("footer") ?>
</body>
</html>
```

Hier ist die selbe Snippetdatei mit Links zu sozialen Medien:

``` php
<div class="footer">
<div class="siteinfo">
<a href="<?php echo $yellow->page->base."/" ?>">&copy; 2017 <?php echo $yellow->page->getHtml("sitename") ?></a>.
<a href="https://twitter.com/datenstromse">Twitter</a>.
<a href="https://github.com/datenstrom">GitHub</a>.
<a href="https://instagram.com/datenstromse">Instagram</a>.
<a href="<?php echo $yellow->page->get("pageEdit") ?>">Edit</a>.
<a href="<?php echo $yellow->text->get("yellowUrl") ?>">Made with Yellow</a>.
</div>
</div>
</div>
<?php echo $yellow->page->getExtra("footer") ?>
</body>
</html>
```

## Sidebar anpassen

Um eine Sidebar anzuzeigen, erstelle die Datei `sidebar.txt` in einem `content`-Verzeichnis. Die Sidebar wird auf allen Seiten im gleichen Verzeichnis angezeigt. Eine andere Sidebar lässt sich in den [Einstellungen](markdown-cheat-sheet-de#einstellungen) ganz oben auf jeder Seite festlegen, zum Beispiel `Sidebar: sidebar`. 

[image screenshot-template3.png Screenshot]

Hier ist eine Beispiel-Sidebar `content/1-home/sidebar.txt`:

```
---
Title: Sidebar
Status: hidden
---
Links

* [Twitter](https://twitter.com/datenstromse)
* [GitHub](https://github.com/datenstrom)
* [Instagram](https://instagram.com/datenstromse)
```
