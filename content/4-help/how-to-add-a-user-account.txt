## Adding user online

You can create a user account in a [web browser](https://github.com/datenstrom/yellow-plugins/tree/master/webinterface):

1. Install a new Yellow version.
2. Click the edit link on a page.
3. Follow instructions to create a user account.

## Adding user via command line

You can add a user account from the [command line](https://github.com/datenstrom/yellow-plugins/tree/master/commandline):

1. Open a terminal window.
2. Go to your Yellow installation, where the `yellow.php` is.
3. Type the following line: `php yellow.php user email@example.com horsebattery`  
4. Now you can log in with `email@example.com` and password `horsebattery`.
