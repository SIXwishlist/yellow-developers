Markdown is a practical way to edit web pages. [Test Markdown](/tests/markdown-format).

## Basics

Formatting text:

    Normal *italic* **bold** `code`

Making a list:

    * item one
    * item two
    * item three

Making an ordered list:

    1. item one
    2. item two
    3. item three

Making a heading:

    # Heading 1
    ## Heading 2
    ### Heading 3

Making quotes:

    > Here's a quote
    >> Here's a quote of a quote
    >>> Here's a quote of a quote of a quote

Making links:

    [Link to page](/help/how-to-make-a-website)  
    [Link to website](https://datenstrom.se)  
    https://en.wikipedia.org/wiki/Markdown

Adding an image:

    ![image](picture.jpg)
    ![image](picture.jpg "Picture")
    ![image](picture.jpg "This is an example image")

Adding an image, here's an alternative format:

    [image picture.jpg]
    [image picture.jpg Picture]
    [image picture.jpg "This is an example image"]

`[slideshare 37321279]` = embed [Slideshare presentation](https://github.com/datenstrom/yellow-plugins/tree/master/slideshare)

`[ea ea-smile]` = show [emoji and colorful images](https://github.com/datenstrom/yellow-plugins/tree/master/emojiawesome)

## Settings

`Title` = page title  
`TitleContent` = page title shown in content  
`TitleNavigation` = page title shown in navigation  
`TitleHeader` = page title shown in web browser  
`Description` = page description  
`Keywords` = page keyword(s), comma separated  
`Author` = page author(s), comma separated  
`Language` = page language  
`Theme` = page theme  
`Template` = page template  
`Parser` = page parser  
`Modified` = page modification date, YYYY-MM-DD format  
`Published` = page publication date, YYYY-MM-DD format  
`Tag` = page tag(s) for categorisation, comma separated  
`Status` = page status for workflow  
`Redirect` = redirect to a new page or URL  
`Navigation` = page navigation  
`Sidebar` = page sidebar  
`Sitename` = name of the website  
`Siteicon` = icon of the website  
`Tagline` = slogan of the website  
`Email` = email of the webmaster
