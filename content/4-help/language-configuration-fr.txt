Comment configurer un site web avec des différents langages.

## Langages

Yellow est multi-langage et il existe [différentes traductions](https://github.com/datenstrom/yellow-plugins/tree/master/language) que vous pouvez télécharger. La langue par défaut est définie dans le fichier `system/config/config.ini`. Une autre langue peut-être définie pour chacune de vos pages dans les paramètres en haut de celle-ci, par exemple `Language: en`.

Voici une page en anglais:

```
---
Title: About us
Language: en
---
Birds of a feather flock together.
```

Une page en allemand:

```
---
Title: Über uns
Language: de
---
Wo zusammenwächst was zusammen gehört.
```

Une page en français:

```
---
Title: À propos de nous
Language: fr
---
Les oiseaux de même plumage volent toujours ensemble
```

## Le mode multi-langage

Si vous souhaitez proposer différentes traductions pour votre site en entier, le meilleur moyen est d'utiliser la fonctionnalité multi-langage de Yellow. Ouvrez le fichier `system/config/config.ini` et modifiez `MultiLanguageMode: 1`. Puis rendez-vous dans votre dossier `content` et créez un dossier pour chaque langue que vous souhaitez rendre disponible. Voici un exemple:

[image screenshot-language1.png Screenshot]

Cette capture d'écran montre les dossiers `1-en`, `2-de` et `3-fr`. Celà donne accès aux URLs suivantes: `http://website/` `http://website/de/` `http://website/fr/` pour respectivement l'anglais, l'allemand et le français. Voici un autre exemple:

[image screenshot-language2.png Screenshot]

Ici nous voyons les dossiers `1-en`, `2-de`, `3-fr` et `default`. Celà donne accès aux URLs suivantes `http://website/en/` `http://website/de/` `http://website/fr/` ainsi qu'une page d'accueil à `http://website/` qui détectera automatiquement la langue pour le visiteur.

Pour mofifier le menu de [navigation](yellow-templates#custom-navigation), ouvrez le fichier `system/config/config.ini` et changez `Navigation: navigation-multilanguage`. Vous pouvez également afficher un moyen pour le visiteur de sélectionner la langue du site ailleurs, par exemple dans le [pied de page](yellow-templates#custom-footer).
