You can customise the functionality of your website.

## Templates

Templates define the functionality of your website. Let's see how templates are made in Yellow. There's a default template defined in file `system/config/config.ini`. A different template can be defined in the [settings](markdown-cheat-sheet#settings) at the top of each page, for example `Template: default`.

Here's an example template file `system/themes/templates/default.html`:

``` php
<?php $yellow->snippet("header") ?>
<?php $yellow->snippet("content-sidebar") ?>
<?php $yellow->snippet("content-main") ?>
<?php $yellow->snippet("footer") ?>
```

Snippets are the building blocks of your website. They are small pieces of [PHP](https://en.wikipedia.org/wiki/PHP) code, written with some help of the [Yellow API](yellow-api). You can put everything into one template, but it's often more practical to split code up into multiple snippets and reuse them. 

Here's an example snippet file `system/themes/snippets/content-main.php`:

``` php
<div class="content main">
<h1><?php echo $yellow->page->getHtml("titleContent") ?></h1>
<?php echo $yellow->page->getContent() ?>
</div>
```

## Custom template

To use a different template set the `Template` setting at the top of a page. You can also create a template with the name of a `content` folder and it will be automatically used. This allows you to generate any [HTML](http://en.wikipedia.org/wiki/HTML) code with Yellow.

Here's an example template file `system/themes/templates/home.html`:

``` php
<?php $yellow->snippet("header") ?>
<?php $yellow->snippet("content-sidebar") ?>
<?php $yellow->snippet("content-home") ?>
<?php $yellow->snippet("footer") ?>
```

Here's an example snippet file `system/themes/snippets/content-home.php`:

``` php
<div class="content main">
<h1><?php echo $yellow->page->getHtml("titleContent") ?></h1>
<?php echo $yellow->page->getContent() ?>
<?php /* Add more PHP code here */ ?>
</div>
```

## Custom navigation

To adjust the navigation open file `system/config/config.ini` and change the `Navigation` setting. A different navigation can be defined in the [settings](markdown-cheat-sheet#settings) at the top of each page, for example `Navigation: navigation`.

The following navigations are available:

`Navigation: navigation` shows top level pages  
`Navigation: navigation-tree` shows dropdown menu  
`Navigation: navigation-sidebar` shows sub level pages in [sidebar](#sidebar)  
`Navigation: navigation-multilanguage` shows a [language selection](language-configuration#multi-language-mode)  
`Navigation: navigation-search` shows a [search bar](https://github.com/datenstrom/yellow-plugins/tree/master/search)  

Here's an example snippet file `system/themes/snippets/navigation.php`:

``` php
<?php $pages = $yellow->pages->top() ?>
<?php $yellow->page->setLastModified($pages->getModified()) ?>
<div class="navigation">
<ul>
<?php foreach($pages as $page): ?>
<li><a<?php echo $page->isActive() ? " class=\"active\"" : "" ?> href="<?php echo $page->getLocation(true) ?>"><?php echo $page->getHtml("titleNavigation") ?></a></li>
<?php endforeach ?>
</ul>
</div>
<div class="navigation-banner"></div>
```

Here's the same snippet file with [social media icons](https://github.com/datenstrom/yellow-plugins/tree/master/fontawesome):

``` php
<?php $pages = $yellow->pages->top() ?>
<?php $yellow->page->setLastModified($pages->getModified()) ?>
<div class="navigation">
<ul>
<?php foreach($pages as $page): ?>
<li><a<?php echo $page->isActive() ? " class=\"active\"" : "" ?> href="<?php echo $page->getLocation(true) ?>"><?php echo $page->getHtml("titleNavigation") ?></a></li>
<?php endforeach ?>
<li><a href="https://twitter.com/datenstromse"><i class="fa fa-twitter"></i></a></li>
<li><a href="https://github.com/datenstrom"><i class="fa fa-github"></i></a></li>
<li><a href="https://instagram.com/datenstromse"><i class="fa fa-instagram"></i></a></li>
</ul>
</div>
<div class="navigation-banner"></div>
```

## Custom footer

To adjust the footer change the corresponding snippet. The footer will be shown on all pages. You can for example show information about your website and more links.

Here's an example snippet file `system/themes/snippets/footer.php`:

``` php
<div class="footer">
<a href="<?php echo $yellow->page->base."/" ?>">&copy; 2016 <?php echo $yellow->page->getHtml("sitename") ?></a>.
<a href="<?php echo $yellow->page->get("pageEdit") ?>">Edit</a>.
<a href="<?php echo $yellow->text->get("yellowUrl") ?>">Made with Yellow</a>.
</div>
</div>
<?php echo $yellow->page->getExtra("footer") ?>
</body>
</html>
```

Here's the same snippet file with social media links:

``` php
<div class="footer">
<a href="https://twitter.com/datenstromse">Twitter</a>.
<a href="https://github.com/datenstrom">GitHub</a>.
<a href="https://instagram.com/datenstromse">Instagram</a>.
<a href="<?php echo $yellow->page->get("pageEdit") ?>">Edit</a>.
<a href="<?php echo $yellow->text->get("yellowUrl") ?>">Made with Yellow</a>.
</div>
</div>
<?php echo $yellow->page->getExtra("footer") ?>
</body>
</html>
```

## Sidebar

To create a sidebar add the file `sidebar.txt` to any `content` folder. The sidebar is automatically shown on all pages in the same folder. A different sidebar can be defined in the [settings](markdown-cheat-sheet#settings) at the top of each page.

Here's an example sidebar file `content/1-home/sidebar.txt`:

```
---
Title: Sidebar
Status: hidden
---
Links

* [Twitter](https://twitter.com/datenstromse)
* [GitHub](https://github.com/datenstrom)
* [Instagram](https://instagram.com/datenstromse)
```
