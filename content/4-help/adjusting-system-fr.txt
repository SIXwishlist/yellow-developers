Tout les paramètres se trouvent dans le dossier `system`. Ici vous pouvez faire des ajustements.

[image screenshot-system.png Screenshot]

Le dossier `config` contient les fichiers de configurations. Le dossier `plugins` contient l'application Yellow. Le dossier `themes` concerne l'aspect visuel de votre site web. Apporter des changements dans ces dossiers affectera entièrement votre site web.

## Plugins et thèmes

Yellow s'installe avec un minimum de fonctionnalité. Vous avez de quoi créer un site web et quelques pages, mais vous souhaitez peut-être intégrer un blog ou une galerie d'images ? Il existe une liste de [plugins](/plugins/) et de [thèmes](/themes/) spécifiques à télécharger.

## Paramètres du système

Le fichier de configuration principal est `system/config/config.ini`. Voici un exemple:

    Sitename: Anna Svensson Design
    Author: Anna Svensson
    Email: anna@svensson.fr
    Language: fr
    Theme: flatsite

Vous pouvez y définir les paramètres par défaut, par exemple le nom du site web et l'email du webmaster. Les [paramètres](markdown-cheat-sheet-fr#paramètres) individuels peuvent être définis au haut de chaque page. Lors d'une nouvelle installation, il est conseillé de changer `Sitename`, `Author` et `Email`.

## Comptes d'utilisateurs

Tous les utilisateurs sont stockés dans le `system/config/user.ini`. Voici un exemple:

    anna@svensson.fr: $2y$10$j26zDnt/xaWxC/eqGKb9p.d6e3pbVENDfRzauTawNCUHHl3CCOIzG,Anna,en,active,946684800,none,/
    demo@demo.com: $2y$10$8TKEuytrwiLSICJNlW5.yOE9CQPXJmqXL2Af6rPsSch1/9UiXEZjm,Yellow,en,active,946684800,none,/
    guest@guest.com: $2y$10$4VENNhK/hb3US8zJdimJduratVuQloACbdmbGhoFUyqHvU6mbpPm.,Guest,en,active,946684800,none,/tests/


L'[interface web](https://github.com/datenstrom/yellow-plugins/tree/master/webinterface) vous permet de créer de nouveaux comptes d'utilisateurs et de changer les mots de passe. Un compte d'utilisateur se compose d'email, mot de passe crypté, différents paramètres de l'utilisateur et à la fin de la ligne est la page d'accueil de l'utilisateur.

Pour restreindre un compte d'utilisateur de définir une page d'accueil pour l'utilisateur. Les utilisateurs ne sont pas autorisés à modifier les pages en dehors de leur propre page d'accueil. Le compte d'utilisateur `Guest` est seulement autorisé à changer `/tests/` pages.
