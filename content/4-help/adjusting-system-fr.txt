Tout les paramètres se trouvent dans le dossier `system`. Vous pouvez y faire des ajustements.

[image screenshot-system.png Screenshot]

Le dossier `config` contient les fichiers de configurations. Le dossier `plugins` contient le logiciel actuel. Le dossier `themes` concerne l'aspect visuel de votre site web. Apporter des changements dans ces dossiers affectera entièrement votre site web.

## Paramètres du système

Le fichier de configuration principal est `system/config/config.ini`. Voici un exemple:

    Sitename: Anna Svensson Design
    Author: Anna Svensson
    Email: anna@svensson.fr
    Language: fr
    Timezone: Europe/Paris
    Theme: flatsite

Vous pouvez y définir les paramètres par défaut, par exemple le nom du site web et l'email du webmaster. Des [paramètres](markdown-cheat-sheet-fr#paramètres) individuels peuvent être définis au haut de chaque page. Lors d'une nouvelle installation, il est conseillé de modifier `Sitename`, `Author` et `Email`.

## Comptes d'utilisateurs

Tous les utilisateurs sont stockés dans le fichier `system/config/user.ini`. Voici un exemple:

    anna@svensson.fr: $2y$10$j26zDnt/xaWxC/eqGKb9p.d6e3pbVENDfRzauTawNCUHHl3CCOIzG,Anna,en,active,946684800,none,/
    demo@demo.com: $2y$10$8TKEuytrwiLSICJNlW5.yOE9CQPXJmqXL2Af6rPsSch1/9UiXEZjm,Demo,en,active,946684800,none,/
    guest@guest.com: $2y$10$4VENNhK/hb3US8zJdimJduratVuQloACbdmbGhoFUyqHvU6mbpPm.,Guest,en,active,946684800,none,/tests/

Le [navigateur web](https://github.com/datenstrom/yellow-plugins/tree/master/edit) vous permet de créer de nouveaux comptes d'utilisateurs et de changer les mots de passe. Un compte d'utilisateur se compose d'un email, d'un mot de passe crypté, de différents paramètres, tandis que le dernier paramètre concerne la page d'accueil de l'utilisateur.

## Plugins et thèmes

Datenstrom Yellow s'installe avec un minimum de fonctionnalité. Vous obtenez les choses les plus importantes pour créer un petit site web et pouvez ajouter d'autres fonctionnalités. 
Il existe des [plugins](/plugins/) et des [thèmes](/themes/) à télécharger. Les plugins suivants sont inclus par défaut:

* [Command](https://github.com/datenstrom/yellow-plugins/tree/master/command): Exécuter des commandes dans une fenêtre de terminal
* [Edit](https://github.com/datenstrom/yellow-plugins/tree/master/edit): Modifiez votre site web dans un navigateur web
* [Image](https://github.com/datenstrom/yellow-plugins/tree/master/image): Intégrer des images et des vignettes
* [Markdown](https://github.com/datenstrom/yellow-plugins/tree/master/markdown): Formatage de texte pour les humains
* [Update](https://github.com/datenstrom/yellow-plugins/tree/master/update): Gardez votre site à jour
