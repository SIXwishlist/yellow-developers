Tout les paramètres se trouvent dans le dossier `system`. Vous pouvez y faire des ajustements.

[image screenshot-system.png Screenshot]

Le dossier `config` contient les fichiers de configurations. Le dossier `plugins` contient l'application Yellow. Le dossier `themes` concerne l'aspect visuel de votre site web. Apporter des changements dans ces dossiers affectera entièrement votre site web.

## Paramètres du système

Le fichier de configuration principal est `system/config/config.ini`. Voici un exemple:

    Sitename: Anna Svensson Design
    Author: Anna Svensson
    Email: anna@svensson.fr
    Language: fr
    Theme: flatsite

Vous pouvez y définir les paramètres par défaut, par exemple le nom du site web et l'email du webmaster. Des [paramètres](markdown-cheat-sheet-fr#paramètres) individuels peuvent être définis au haut de chaque page. Lors d'une nouvelle installation, il est conseillé de modifier `Sitename`, `Author` et `Email`. Vous pouvez y [définir le langage](language-configuration-fr) et installer d'autres traductions.

## Comptes d'utilisateurs

Tous les utilisateurs sont stockés dans le fichier `system/config/user.ini`. Voici un exemple:

    anna@svensson.fr: $2y$10$j26zDnt/xaWxC/eqGKb9p.d6e3pbVENDfRzauTawNCUHHl3CCOIzG,Anna,en,active,946684800,none,/
    demo@demo.com: $2y$10$8TKEuytrwiLSICJNlW5.yOE9CQPXJmqXL2Af6rPsSch1/9UiXEZjm,Yellow,en,active,946684800,none,/
    guest@guest.com: $2y$10$4VENNhK/hb3US8zJdimJduratVuQloACbdmbGhoFUyqHvU6mbpPm.,Guest,en,active,946684800,none,/tests/

L'[interface web](https://github.com/datenstrom/yellow-plugins/tree/master/webinterface) vous permet de créer de nouveaux comptes d'utilisateurs et de changer les mots de passe. Un compte d'utilisateur se compose d'un email, d'un mot de passe crypté, de différents paramètres, tandis que le dernier paramètre concerne la page d'accueil de l'utilisateur. Vous pouvez en effet [restreindre l'accès de votre site web](security-configuration-fr#user-restrictions) en définissant une page d'accueil propre à chaque utilisateur.

## Plugins et thèmes

Yellow s'installe avec un minimum de fonctionnalité. Vous avez de quoi créer un site web et quelques pages, mais vous souhaitez peut-être intégrer un blog ou une galerie d'images ? Il existe une liste de [plugins](/plugins/) et de [thèmes](/themes/) spécifiques à télécharger.
