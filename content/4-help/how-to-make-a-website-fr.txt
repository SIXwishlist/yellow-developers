Apprenez à faire un site web avec Yellow.

[image screenshot-website.jpg Screenshot]

## Installer votre site

1. [Téléchargez et dézippez Yellow](https://github.com/datenstrom/yellow/archive/master.zip).
2. Copiez tout les fichiers chez votre hébergeur.
3. Accédez à votre site depuis un navigateur web.

Votre site web est immédiatement accessible. L'installation est fournie avec deux pages, 'Home' et 'About'. Elles sont juste un exemple pour commencer, vous pouvez les modifier comme il vous convient. Vous pouvez créer votre site web en ajoutant des fichiers et des dossiers.

S'il y a des problèmes, vérifiez la [configuration du serveur web](web-server-configuration-fr) et les [paramètres du système](adjusting-system-fr#paramètres-du-système).

## Écrire une page web

Jetons un oeil dans le dossier `content`, où se trouvent toutes vos pages web. Ouvrer le fichier `content/1-home/page.txt`. Vous y verrez les paramètres et le texte de la page. Vous pouvez changer le titre de la page `Title` ainsi que d'autres [paramètres](markdown-cheat-sheet-fr#paramètres) en haut de la page. Voici un exemple:

```
---
Title: Home
---
Votre site web fonctionne!

Vous pouvez [edit - modifier cette page] ou utiliser un éditeur de texte.
```

Pour créer une nouvelle page, ajoutez un nouveau fichier dans le dossier `1-home` ou dans tout autre dossier du dossier `content`. Pour ajouter votre page au menu de navigation, créez un dossier avec un préfixe numérique. Seuls les dossiers avec un préfixe sont présentés dans la navigation. Voici un autre exemple de page:

```
---
Title: Yellow est fait pour les gens qui font des sites web
---
Vous pouvez [edit - modifier cette page]. L'aide vous donne plus 
d'informations sur la façon de créer de petites pages web, blogs et wikis.
```

Maintenant ajoutons une image avec le [plugin image](https://github.com/datenstrom/yellow-plugins/tree/master/image):

```
---
Title: Yellow est fait pour les gens qui font des sites web
---
[image picture.jpg]

Vous pouvez [edit - modifier cette page]. L'aide vous donne plus 
d'informations sur la façon de créer de petites pages web, blogs et wikis.
```

Voyons comment créer un titre différent dans le menu de navigation:

```
---
Title: Yellow est fait pour les gens qui font des sites web
TitleNavigation: Home
---
[image picture.jpg]

Vous pouvez [edit - modifier cette page]. L'aide vous donne plus 
d'informations sur la façon de créer de petites pages web, blogs et wikis.
```

Vous pouvez écrire vos pages en [Markdown](markdown-cheat-sheet-fr)

## Créer une sidebar

Pour créer une sidebar (menu latéral), ajoutez le fichier `sidebar.txt` dans un dossier de `content`. La sidebar est automatiquement visible sur toutes les pages présentes dans le même dossier. Voici un exemple de sidebar:

```
---
Title: Sidebar
Status: hidden
---
Liens

* [Twitter](https://twitter.com/datenstromse)
* [GitHub](https://github.com/datenstrom)
```
